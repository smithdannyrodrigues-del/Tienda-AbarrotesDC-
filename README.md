# Tienda-AbarrotesDC-
Base de datos en SQL diseñada para digitalizar y gestionar los registros de crédito y cuentas por cobrar de una tienda retail

##  Objetivo del Proyecto
Este proyecto nace con el propósito de ofrecer una solución tecnológica a un problema real y muy común: la dependencia de métodos manuales (como los cuadernos físicos) para registrar créditos o "fiados" en los comercios locales. El objetivo es estructurar una base de datos que permita digitalizar estos registros, asegurando que la información de las deudas esté siempre segura, precisa y accesible, optimizando así la gestión del negocio.

##  Características Principales
- **Registro de Clientes:** Estructura para gestionar la información de los usuarios que solicitan crédito.
- **Control de Deudas:** Seguimiento detallado de los montos adeudados, productos y fechas.
- **Historial de Abonos:** Tablas diseñadas para registrar los pagos parciales y mantener el saldo actualizado.

##  Tecnologías y Enfoque
- **Lenguaje:** SQL
- **Enfoque:** Digitalización de registros, integridad de datos y mejora continua de procesos comerciales.
